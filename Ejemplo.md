# Cientificos descubren metodo para construir computadoras cuanticas en masa
## Que son las computadoras cuanticas? 
Las computadoras cuanticas son computadoras cuya base de software y de codigo se basa
en **bits cuanticos** *(o Qubits)*, a diferencia de las computadoras actuales las cuales utilizan bits.

Diferencias entre ambos tipos de dato:
- Los bits solo pueden estar 1 o 0
- los qubits pueden ser 1, 0, ambos simultaneamente o cualquier combinacion que la cuantica permita.
- Aunque los qubits permiten una complejidas significativamente superior; son notablemente más dificiles de mantener
